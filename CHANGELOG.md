# Changelog

All notable changes to this project are documented here. The format is based on
[Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [1.0.0] - 2026-06-02

### Added
- Initial public open-source release.
- macOS desktop app (Electron): floating always-on-top widget, screenshot + OCR,
  clipboard monitoring, global hotkeys, system-tray integration.
- Chrome extension (Manifest V3): text-selection detection, injected widget,
  right-click context menu.
- AI reply generation with OpenAI GPT: message-type and tone detection, quick
  refinements (shorter / formal / casual / friendly).
- Bring-your-own API key, stored locally. No embedded key, no telemetry.
